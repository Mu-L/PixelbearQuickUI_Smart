## 介绍

这里正式开发一款基于 UE5 + QuickUI开发的可配置通用交互模板，使用数据驱动的方式来配置 UI 界面，使得 UI 界面可以根据数据的变化而自动更新，同时也可以根据数据的变化来自动生成 UI 界面。

## 下载与使用

因为使用git 的lfs 来管理项目资源，所以需要先安装git lfs 插件，具体安装方法可以参考 [git lfs 安装](https://git-lfs.github.com/)

### 项目下载

```bash
git clone https://github.com/Pixelbear/PixelbearQuickUI_Smart_UE54.git
```

或者云盘下载 [百度云盘下载](https://pan.baidu.com/s/16pslO7yXKvi9YCJQTaM8uQ?pwd=9699)

### 使用方法

1. 安装 QuickUI 插件。
   - 解压下载的 QuickUI 插件压缩包。
   - 拷贝 QuickUI 插件文件夹到`PixelbearQuickUI_Smart_UE54/Plugins`目录下。
2. 等待项目加载完成，打开 `PixelbearQuickUI.uproject` 文件。
3. 使用教程参考 [QuickUI 开发日合集](https://space.bilibili.com/74228375/lists/5958458)
